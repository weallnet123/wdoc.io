RenderCustomNestedScrollViewViewport
description

RenderCustomNestedScrollViewViewport class Null safety

The RenderViewport variant used by CustomNestedScrollView.

This viewport takes a SliverOverlapAbsorberHandle and notifies it any time the viewport needs to recompute its layout (e.g. when it is scrolled).
